# MP1

## Basic Ray Tracer

Supported Objects:

* Spheres

* Planes (and checkered planes)

* Triangles

Supported Light Types:

* Ambient lights

* Directional lights

Lighting (via Phong Reflection Model):

* Ambient

* Diffuse

* Specular

* HDR tone-mapping via gamma-compression

Projections:

* Orthographic

* Perspective

Other Goodies:

* Arbitrary image size support

* Anti-aliasing support (arbitrary positive integers)

* Multithread support

* Specify output file
